# create-ai-images-in-Hebrew
A web application that uses an API of DALL-E from openAi and translation. and creates Ai images according to instructions in Hebrew.

# Client :
- React.js
- fetch
- Axios
- Styled components

# Server :
- Node.js
- Express
- Mongo.db
- mongoose
- dotenv
- file-saver

# Images :
- cloudinary

# Api :
- openai
- free translating
